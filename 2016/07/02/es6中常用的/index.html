es6 中比较常用的语法

发表于 2016-07-02 | 分类于 es6 | | 阅读次数

1. let

  • 使用 let 定义变量不能重复定义，否则报错
  • let 定义的变量没有变量提升
  • 块级作用域(只在定义的当前代码块有效)
    • 在 es6 之前,如果使用 for 循环,定义的 i 在循环之后,会变成全局变量,使用 let 可以很好的解决

2. const

  • 使用 const 定义常量不能重复定义，否则报错
  • const 定义的变量没有变量提升
  • 块级作用域(只在定义的当前代码块有效)
  • 定义的同时必须赋值，否则报错

2.1 let 和 const 不同

  • 定义的同时必须赋值，否则报错
  • const 用来定义常量, let 用来定义变量

3. String

  • 如果有字符串拼接, 就使用 es6 模板字符串–用2个`( Esc 键下面的)将要拼接的字符串包起来
  • 如果没有字符串拼接, 还是用 ‘ 或者 “

4. Array

  • Array.from() 将伪数组变成真数组
    • 处理 dom 时常用, 比如querySelectorAll() 获取的是伪数组对象
  • Array.prototype.find() 数组实例上的方法, 即数组原型上的方法, 找出第一个符合条件的数组成员并返回, 没找到返回 undefined
  • Array.prototype.includes()返回一个布尔值, 表示某个数组是否包含给定的值

5. 箭头函数-lambda 表达式

  • 如果箭头函数不需要参数或需要多个参数, 就使用一个圆括号代表参数部分
  • 如果箭头函数的代码块部分等于一条语句, 可以省略大括号, 并且箭头后面表达式就是返回值
  • 如果箭头函数的代码块部分多于一条语句, 就要使用大括号将它们括起来, 并且使用 return 语句返回
  • 箭头函数中的 this 指向的就是定义的环境

参考: ECMAScript 6 入门
